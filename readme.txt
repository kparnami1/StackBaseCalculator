how to run and compile multi_thread java Program with Command Prompt

1)Open Command Prompt and go to directory where java file is located "cd Desktop"
2)Run Java file fot compilation as "javac StackCalculator.java"
3)Type with arguements like "java StackCalculator
4)The program will ask the user "Enter an arithmetic expression in infix notation" FOr Example 80+80
5)Output will be:
   Overflow occurred during addition
   Postfix expression: 8080+
   Result: -96
